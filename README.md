# 남성 수제구두 쇼핑몰 HANZO


## :mans_shoe: 프로젝트 소개

- 남성 수제구두를 판매하는 간단한 웹사이트입니다. 사용자는 판매하고 있는 신발 목록을 볼 수 있고 원하는 신발을 선택해 구매할 수 있습니다.

<br>

## :books: 사용 기술

-  Html / Css / JavaScript / Jsp
-  Oracle

<br>

## :small_blue_diamond: 주요 기능

1. 회원가입 / 로그인
2. 원하는 상품 선택해 구매 기능
3. 마이페이지 기능
   - 회원가입 시 입력했던 내 정보 보기
   - 현재까지 내가 구매한 상품 갯수 및 목록 확인
4. 게시판 기능 ( 게시글 쓰기 및 댓글 작성 )
